# TeraCast API 🎬

Professional TeraBox Video Streaming & Downloader API

## Features ✨

- 🎥 Extract file information from TeraBox share links
- 📥 Get direct download links
- 🎬 HLS streaming support (M3U8 playlists)
- 🔐 Password-protected link support
- ⚡ Async/await for high performance
- 🔄 Automatic caching
- ️ Rate limiting
- 📱 Mobile-friendly
